# dashgo
This project was made for study purposes, possible due to [Rocketseat](https://www.rocketseat.com.br/)'s Ignite course.

## Technologies used
- Next.js
- TypeScript
- Chakra UI
- React Hook Form
- Yup
- Axios
- React Query
- Jest & Testing Library
- Faker
- MirageJS
